優選IP訂閱生成器

只做三件事：導入自建節點、批量替換爲優選 IP、輸出可導入主流客戶端的訂閱鏈接。

適合場景

3X-UI / Xray / VPS 自建節點 + 你自己找好的 Cloudflare 優選 IP。

重點兼容自建 VMess / VLESS / Trojan 的 WS + TLS 常見寫法。

支持 IP、IP:端口、域名、域名:端口，# 後面可寫備註。

說明

  • 不會幫你找優選 IP，只負責把你現成的優選 IP 替換進節點。
  • 生成的鏈接是“同一份節點內容”的不同導出格式，分別給不同客戶端使用。
  • 推薦勾選“保留原 Host / SNI”，這樣更適合 Cloudflare CDN 場景。
  • 訂閱鏈接裏的節點信息會通過服務端密鑰加密，請不要把鏈接公開分享。

客戶端怎麼用

  • V2rayN： 用原始訂閱鏈接
  • Shadowrocket： 用原始訂閱鏈接
  • Clash / Mihomo： 用 Clash 訂閱鏈接
  • Surge： 用 Surge 訂閱鏈接
